
# and-exra-subscription

このプロジェクトは、WIXからアクセスされる6つの定期購入グループ（g1〜g6）に対応し、
それぞれの送料・発送日・割引条件をサーバー側で処理するためのExpressサーバーです。

## グループ構成
| グループ | 発送日 | 地域         | 送料   |
|----------|--------|--------------|--------|
| g1       | 5日    | 本州         | ¥1,100 |
| g2       | 5日    | 北海道/沖縄 | ¥3,300 |
| g3       | 15日   | 本州         | ¥1,100 |
| g4       | 15日   | 北海道/沖縄 | ¥3,300 |
| g5       | 25日   | 本州         | ¥1,100 |
| g6       | 25日   | 北海道/沖縄 | ¥3,300 |
